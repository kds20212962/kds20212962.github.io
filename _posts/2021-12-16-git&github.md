# 깃이란?
분산형 버전 관리 시스템(VCS)이다.

깃은 다양한 명령어를 통해 이루어진다.

* git add
* git coomit
* git push
* git pull


# 깃허브?
 분산 버전 관리 툴인 깃 (Git)저장소 호스팅을 지원하는 웹 서비스이다
Local 저장소에서 원격 저장소로 파일을 저장하게 되는데 그 때 원격 저장소를 깃허브에서 맡고 있다.
